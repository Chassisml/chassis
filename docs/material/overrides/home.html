What is Chassis?

Chassis is an open-source project that automatically turns your ML models into production containers.

Chassis picks up right where your training code leaves off and builds containers for any target architecture. Execute a single Chassis job then run your containerized models in the cloud, on-prem, or on edge devices ( Raspberry Pi, NVIDIA Jetson Nano, Intel NUC, and others).
